В папке lesson_19, создать небольшой проект используя react, redux
Должен присутствовать input и кнопка добавить, нажатие на которую должно добавлять
в список новую строку (значение input-a), список отобразить ниже.
* Очищать значение input-a при добавлении

Возможные элементы проекта:
- главный React-компонент
- Redux store, listReducer (массив)
- Action 

Создать middleware “deduplicator”, который “остановит” action создающий строку со значением которое уже создано.

Создать middleware storage который будет записывать список в localStorage.

* При загрузке страницы брать последнюю версию списка из localStorage
