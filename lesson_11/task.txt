Создать маленькое веб приложение которое отображает на основе публичного API 

  Список требований:

    1. Отобразить надпись что приложение ищет данные
    2. Найти и использовать публичное API для получения IP
    3.Используя найденный IP, и https://jsonbin.io/api-reference/geoip/lookup, получить данные о стране и городе, отобразить их на экране. Если произошла ошибка, вывести соответствующее сообщение на экране
    4*. Для получения IP адреса использовать более одного источника, и ожидать самый быстрый ответ
