  2. Создать роут /contacts где будет надпись “Our contacts”
  3. Создать роут для ненайденной страницы, где будет надпись “404”, 
  4. Создать роут /color/:color, на котором цвет фона сайта будет из url-а, например на /color/grey фон должен быть grey

Part 3 * HOC
  Создать HOC компонент ghostHOC, который после клика на обернутый компонент - на 3 секунды сделает его полупрозрачным и недоступным для клика.
  Пример использования:

    function myButton(props) {
      return <button {...props}>{props.children}</button>
    }

    const GhostButton = ghostHOC(myButton);
    const GhostLink = ghostHOC(Link);
