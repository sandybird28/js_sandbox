В файл color.js скопируйте код класса Color из соответствующего домашнего задания

В файлах index.html, index.js

Стили необходимые для этого задания можно писать в css/sass/less файле
В файле index.html создать div с каким-то id, внутри него создать несколько img с изображениями
В файле index.js создать класс MiniSlider, который принимает в аргумент конструктора id элемента внутри
которого должны быть img, назовем элемент с этим id - корневым элементом.
Создать метод hideAll(), который скроет все изображения в корневом элементе, вызвать его в конструкторе.
Создать метод show(n), который покажет изображение под индексом n в корневом элементе, использовать его в
конструкторе для показа первого изображения.
Создать метод createButtons(), который создаст кнопки Prev и Next (они должны располагаться слева и справа от
изображения), использовать его в конструкторе.
Создать метод next(), который будет вызываться по нажатию на кнопку Next и показывать следующее изображение
(после последнего показывать первое)
Создать метод prev(), аналогичный методу next()
Создать метод destroy(), который удалит добавленные кнопки и покажет все изображения, другими словами “зачистит следы своего пребывания”
* В файл index.js импортировать класс Color и c помощью его методов random и toString, задавать
  случайный цвет рамке кнопок Prev и Next при нажатии (новый цвет при каждом нажатии)
